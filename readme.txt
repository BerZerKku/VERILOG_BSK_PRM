Plis_Bsk_Prm_v2_04 
xx.xx.2018

Прошивка ПЛИС Prm для платы АВАНТ BSK_023 (DD14 Altera EPM7128STC). Написана на Verilog, за основу взята прошивка v2.3.

Для блока БСК 01-16 адрес CS = 0111 (все тесты сделаны для него).
Для блока БСК 17-32 адрес CS = 0101. 

Quartus 13.0 sp1
python 2.7.14
vunit-hdl (3.0.2)

"run.py" - запуск VUnit тестов с выводом информации в консоль.
"run.bat" - запуск VUnit тестов в отладчике (ModelSim).
